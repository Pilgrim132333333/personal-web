# 郭梓耕个人介绍网站

这是一个深色风格的静态个人介绍网站。它不再以简历卡片为主，而是用简历内容作为素材，讲清楚个人兴趣、思考方式、项目背后的问题、学习路径和联系方式。

## 文件结构

- `index.html`: 页面骨架，包含首屏、个人线索、介绍、故事、工作台、路径、工具箱与联系方式
- `styles.css`: 深色视觉系统、响应式布局和展开面板样式
- `script.js`: 顶部导航、移动端菜单、中英文切换、本地语言记忆和展开内容渲染
- `assets/hero-studio.png`: 首屏背景图

## 当前特点

- 内容定位是“自我介绍网站”，不是简历复刻。
- 每个主要内容项都支持展开阅读更多细节。
- 项目经历被改写为“项目背后的问题、负责内容和收获”。
- 实习、教育、志愿经历被放入个人成长路径中。
- 支持中文和英文两个语言版本，并会记住用户选择。
