# 郭梓耕个人展示网站

这是一个基于个人中文简历内容整理的深色风格静态作品集网站，可以直接用浏览器打开 `index.html` 预览。

## 文件结构

- `index.html`: 页面内容与结构，包含个人概览、经历、项目、技能、教育与联系方式
- `styles.css`: 深色视觉系统与响应式布局
- `script.js`: 顶部导航状态、移动端菜单交互、中英文语言切换与本地语言记忆
- `assets/hero-studio.png`: 首屏背景图

## 内容来源

页面内容已根据 `ZigengGuo_chinese.docx` 更新，重点突出：

- 宁波诺丁汉大学计算机科学背景
- 软件开发、全栈开发与 AI 应用实习方向
- 联想 GSG E2E 部门实习经历
- AI 学习助手、遗传算法、3D AI Agent、叶子识别与视觉推理项目
- Java、Python、C++、Vue.js、Spring Boot、MySQL、机器学习和计算机视觉技能

## 双语支持

网站支持中文和英文两个语言版本。首次访问会根据浏览器语言自动选择中文或英文；用户手动切换后，语言偏好会保存在浏览器本地。
